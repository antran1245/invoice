# Objective

Create a invoice application.

## Run the application

Start

`yarn start` or `npm start`

# Packages

- React (CRA -> creat-react-app)
- TypeScript
- Sass
- react-bootstrap
- bootstrap
- fontawesome

# Features

- Light/Dark theme mode switch
- Sidebar Nav
- Edit invoice overlay
